# Travel Scripts
> Useful scripts for planning trips

## Setup

### Setup API Keys

```
cp .env.example .env
vim .env
```

### Build and link

```
npm link
```

### Development

```
npm run create-travel-cards
```

## Scripts

### `create-trello-cards`

Create trello cards for tourist attractions that contain:

- Hours
- Address
- Telephone number
- Website
- Google Places Link
- Up to 10 photo attachments
